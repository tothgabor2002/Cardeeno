Application Maker Prompt
Create a clean, visually appealing word-pair matching game for vocabulary learning.
Game Requirements:
* The interface shows two vertical columns of word cards:
   * Left column: Known language (e.g., English)
   * Right column: Learning language (e.g., German)
* Display word-pair cards in random order, falling or appearing in varied positions.
* Mix the pairs so the correct matches do not appear on the same row or in predictable alignment.
* The user matches words by:
   1. Clicking a card on one side
   2. Clicking the corresponding card on the other side
* If the two selected cards form a correct pair:
   * Both cards disappear with a success animation
   * New word pairs smoothly and slowly appear in the empty spots
* Continue presenting randomized word pairs to encourage active learning and pattern recognition.
